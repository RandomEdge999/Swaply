# Swaply Frontend

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run Development Server:
   ```bash
   npm run dev
   ```

## Features

- **Auth**: Login and Register pages.
- **Browse**: View available games.
- **Dashboard**: Manage rentals and view profile.
- **List Game**: Create new game listings.

## Configuration

- API URL is hardcoded to `http://localhost:4000` for MVP.
